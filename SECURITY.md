# Security Policy

## Supported Versions

This project is pre-alpha and has no stable supported release yet.

## Reporting a Vulnerability

Open a GitHub security advisory or issue after the repository becomes public.

Do not include secrets, API keys, recordings, or private transcripts in public
issues.

## Sensitive Data

The application may process dictated text that contains private or confidential
information. The default design goal is local processing and minimal logging.
